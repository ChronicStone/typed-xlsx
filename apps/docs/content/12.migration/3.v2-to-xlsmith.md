The package has been renamed from `@chronicstone/typed-xlsx` to `xlsmith`.

There are no API changes in this migration. The only required change is the import path.

```ts
// before
import { createExcelSchema, createWorkbook, createWorkbookStream } from "@chronicstone/typed-xlsx";

// after
import { createExcelSchema, createWorkbook, createWorkbookStream } from "xlsmith";
```

If you publish templates, examples, or internal packages that depend on the old package name, update their dependency list and import specifiers at the same time.

The old package is deprecated on npm and points to `xlsmith`.
