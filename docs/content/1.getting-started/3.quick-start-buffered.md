This guide produces a real-world invoice report with formula columns and a summary row — the kinds of things you'd actually ship to a finance team.

## Install

```bash
npm install @chronicstone/typed-xlsx
```

## Define your row type

```ts twoslash
type Invoice = {
  invoiceId: string;
  customer: string;
  issueDate: string;
  qty: number;
  unitPrice: number;
  taxRate: number;
  status: "paid" | "pending" | "overdue";
};
```

## Build the schema

```ts twoslash
type Invoice = {
  invoiceId: string;
  customer: string;
  issueDate: string;
  qty: number;
  unitPrice: number;
  taxRate: number;
  status: "paid" | "pending" | "overdue";
};
// ---cut---
import { createExcelSchema } from "@chronicstone/typed-xlsx";

const invoiceSchema = createExcelSchema<Invoice>()
  .column("invoiceId", { header: "Invoice #", accessor: "invoiceId", width: 14 })
  .column("customer", { header: "Customer", accessor: "customer", minWidth: 20 })
  .column("issueDate", { header: "Issue Date", accessor: "issueDate", width: 14 })
  .column("qty", { header: "Qty", accessor: "qty", width: 8 })
  .column("unitPrice", {
    header: "Unit Price",
    accessor: "unitPrice",
    width: 14,
    style: { numFmt: "$#,##0.00" },
  })
  // Formula column: Qty × Unit Price, rounded to 2 dp
  .column("subtotal", {
    header: "Subtotal",
    formula: ({ row, fx }) => fx.round(row.ref("qty").mul(row.ref("unitPrice")), 2),
    width: 14,
    style: { numFmt: "$#,##0.00" },
    // Summary: live Excel SUM formula over the column range
    summary: (s) => [s.formula("sum")],
  })
  .column("taxRate", {
    header: "Tax %",
    accessor: "taxRate",
    width: 10,
    style: { numFmt: "0.0%" },
  })
  // Formula column: Subtotal × Tax Rate
  .column("taxAmount", {
    header: "Tax",
    formula: ({ row, fx }) => fx.round(row.ref("subtotal").mul(row.ref("taxRate")), 2),
    width: 12,
    style: { numFmt: "$#,##0.00" },
    summary: (s) => [s.formula("sum")],
  })
  // Formula column: Subtotal + Tax
  .column("total", {
    header: "Total",
    formula: ({ row }) => row.ref("subtotal").add(row.ref("taxAmount")),
    width: 14,
    style: { numFmt: "$#,##0.00" },
    summary: (s) => [s.formula("sum")],
  })
  .column("status", {
    header: "Status",
    accessor: "status",
    width: 12,
    // Conditional styling per row
    style: (row) => ({
      font: {
        bold: row.status === "overdue",
        color: {
          rgb: row.status === "paid" ? "166534" : row.status === "overdue" ? "B42318" : "92400E",
        },
      },
    }),
  })
  .build();
```

## Create the workbook and export

```ts twoslash
type Invoice = {
  invoiceId: string;
  customer: string;
  issueDate: string;
  qty: number;
  unitPrice: number;
  taxRate: number;
  status: "paid" | "pending" | "overdue";
};
import { createExcelSchema } from "@chronicstone/typed-xlsx";
const invoiceSchema = createExcelSchema<Invoice>()
  .column("invoiceId", { accessor: "invoiceId" })
  .column("customer", { accessor: "customer" })
  .column("issueDate", { accessor: "issueDate" })
  .column("qty", { accessor: "qty" })
  .column("unitPrice", { accessor: "unitPrice" })
  .column("subtotal", {
    formula: ({ row, fx }) => fx.round(row.ref("qty").mul(row.ref("unitPrice")), 2),
  })
  .column("taxRate", { accessor: "taxRate" })
  .column("taxAmount", {
    formula: ({ row, fx }) => fx.round(row.ref("subtotal").mul(row.ref("taxRate")), 2),
  })
  .column("total", { formula: ({ row }) => row.ref("subtotal").add(row.ref("taxAmount")) })
  .column("status", { accessor: "status" })
  .build();
// ---cut---
import { createWorkbook } from "@chronicstone/typed-xlsx";

const rows: Invoice[] = [
  {
    invoiceId: "INV-001",
    customer: "Acme Corp",
    issueDate: "2025-01-15",
    qty: 10,
    unitPrice: 299,
    taxRate: 0.2,
    status: "paid",
  },
  {
    invoiceId: "INV-002",
    customer: "Globex Inc",
    issueDate: "2025-01-22",
    qty: 5,
    unitPrice: 850,
    taxRate: 0.2,
    status: "pending",
  },
  {
    invoiceId: "INV-003",
    customer: "Initech",
    issueDate: "2025-02-03",
    qty: 2,
    unitPrice: 1200,
    taxRate: 0.2,
    status: "overdue",
  },
];

const workbook = createWorkbook();
workbook.sheet("Invoices", { freezePane: { rows: 1 } }).table("invoices", {
  schema: invoiceSchema,
  rows,
  title: "Invoice Report — Q1 2025",
});

// Write to disk
await workbook.writeToFile("./invoices.xlsx");

// Or get a Buffer for HTTP responses, S3 uploads, etc.
const buffer = workbook.toBuffer();
```

## What you get

The exported file has:

- Column headers with the labels you specified
- Formula columns that Excel evaluates live (for this layout, formulas like `=ROUND(D2*E2,2)`, `=ROUND(F2*G2,2)`, and `=F2+H2`)
- A summary row below the data with range formulas for subtotal, tax, and total
- Conditional text colour on the status column (green / amber / red)
- A frozen header row so scrolling keeps the column labels visible
